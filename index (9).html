Single Case Agreements
State insurance regulators, timely access laws, and complaint portals — all 50 states
What is an SCA? A Single Case Agreement (SCA) is a temporary contract allowing a patient to receive out-of-network care at in-network rates when no in-network provider is available within timely access standards. If an insurer denies an SCA request, file a complaint with the state insurance commissioner using the links below.

Key federal leverage: The Mental Health Parity and Addiction Equity Act (MHPAEA) and network adequacy regulations require insurers to ensure timely access to behavioral health services. Use the state law citations below to cite statutory authority in denial appeals.
State Insurance Dept / Commissioner Timely Access Law File Complaint Notes
* Dept links go to official state insurance department. Complaint portals go directly to the consumer complaint form where available.
